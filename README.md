## 🔥 Fire Hazard Intelligence Platform

A web-based decision support tool designed to help firefighters assess and respond to fire risks more effectively. The platform visualises hazard data on an interactive map, combining environmental, infrastructural, and social factors to highlight high-risk zones before and during incidents.

### 🚒 Key Features

* **Interactive Hazard Map**
  Visualises fire risk zones with dynamic scoring and colour-coded severity levels.

* **Multi-Factor Risk Analysis**
  Combines building vulnerability, lithium devices, human factors, infrastructure, and chemical proximity into a composite risk score.

* **AI-Powered Risk Overview**
  Generates summaries to support faster decision-making.

* **Operational Awareness**
  Displays nearby fire stations and key infrastructure.

* **Educational Insights**
  Highlights modern fire risks such as lithium battery hazards.

### 🧠 Purpose

To bridge the gap between data and real-world emergency response by providing clear, actionable insights for firefighters.

### 🛠️ Tech Stack

* React + Leaflet
* FastAPI
